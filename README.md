# Todo App

Multi user todo application

## Requirement
- PHP 7.4+
- Laravel 7.2.5+

## Quick Start
- `composer install`
- create `.env` file
- `php artisan key:generate`
- `php artisan migrate`
- `php artisan db:seed`
